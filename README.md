# Nega Nina — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
A Nega Nina é uma loja de vestuário feminino que atua no bairro de Lagoa Nova em Natal/RN. É conhecida por sua curadoria refinada de peças de alta costura, alfaiataria contemporânea e vestidos ideais para o dia a dia e ocasiões especiais.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout minimalista e transições modernas de componentes.
- https://github.com/tailwindlabs/tailwindui-templates — Padrões premium de design system com Tailwind CSS.
